# Модуль lifecycle-demo

Демонстрационный модуль для отслеживания всех хуков жизненного цикла Nuxt модулей.

## Что делает модуль

Модуль логирует все ключевые хуки жизненного цикла модулей:

- **M0: MODULE SETUP** — инициализация модуля (лёгкие вещи в `setup()`)
- **M1: modules:done** — все модули установлены
- **M2: ready** — Nuxt готов к использованию (тяжёлые операции, добавление плагинов)
- **M3: build:done** — сборка завершена
- **M4: close** — процесс завершается (очистка ресурсов)

Также модуль добавляет runtime-плагины:
- **M-RT-C**: клиентский runtime-плагин
- **M-RT-S**: серверный runtime-плагин

## Как использовать

Модуль уже подключен в `nuxt.config.ts`. Запустите:

```bash
npm run dev
```

Вы увидите логи модуля в консоли при старте dev-сервера и при сборке.
